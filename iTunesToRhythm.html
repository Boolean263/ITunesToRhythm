iTunesToRhythm

This little piece of insanitym is a tool that transfers your iTunes ratings and playcounts into your RhythmBox library.

Prerequesites:

  • Rhythmbox
  • Python
  • libxml2
  • python-libxml2 (Python bindings for libxml2)
If you are running a desktop distribution of linux, you probably already have these installed.

Execution:

  1. Download the follwoing 3 files and put theme somewhere:
    • dumpitunes.py
    • dumprhythm.py
    • iTunesToRhythm.py
  2. Move/copy/mount all your music where RhythmBox can find them (the folder structure doesn't matter)
  3. Copy or mount your "iTunes Music Library.xml" file to linux
  4. Execute the following command: python iTunesToRhythm.py -w -d ITunes\ Music\ Library.xml ~/.local/share/rhythmbox/rhythmdb.xml
  5. Manually resolve any ambiguities your are prompted for
  6. Enjoy your music

Usage:

Usage: iTunesToRhythm [options] <path to ItunesMusicLibrary.xml> <path to rhythmdb.xml>

Options:
  -h, --help          show this help message and exit
  -c, --confirm       confirm every match
  -w, --writechanges  write changes to destination file
  -d, --disambiguate  prompt user to resolve ambiguities
  -l, --fastandloose  ignore differences in files name when a file size match
                      is made against  a single song.   Will not resolve
                      multiple matches
